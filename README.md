# Credit Scoring Model

##  Project Overview

This project predicts whether a customer has **Good Credit Risk** or **Bad Credit Risk** using Machine Learning classification algorithms. The model is trained on customer financial information such as age, housing, savings account, checking account, loan amount, loan duration, and loan purpose.

This project was developed as part of the **CodeAlpha Machine Learning Internship**.

---

##  Objective

To build a machine learning model that predicts an individual's creditworthiness based on financial and personal information.

---

##  Dataset

The dataset contains the following features:

- Age
- Sex
- Job
- Housing
- Saving Accounts
- Checking Account
- Credit Amount
- Duration
- Purpose
- Risk (Target Variable)
